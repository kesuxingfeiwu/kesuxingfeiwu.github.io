使用Hexo搭建Github个人博客

  1. 安装node.js

  2. 安装Git

  3. 安装hexo

    创建一个文件夹，用于存放Github仓库文件。

    右键文件夹，选择Git Bash Here打开，进入Git命令窗口。

    为保障网络稳定，使用国内淘宝官方的服务器。安装hexo时，因为是从国外服务器下载，受网络影响较大，可能出现异常，因此使用国内淘宝官方的服务器安装。后续安装时，命令以cnpm开头而不是npm。

    npm install -g cnpm --registry=https://registry.npm.taobao.org

    安装hexo

    cnpm install -g hexo-cli

    初始化hexo

    hexo init

    hexo初始化时，出现报错：文件夹禁止访问，可能是文件夹权限不足导致，可以在属性中将文件夹的最高权限打开。

    这时创建的文件夹下会多出许多文件。

    输入hexo s后，在浏览器输入localhost:4000，可以看到一篇hexo自带的Hello World博客。

    hexo s

    在cmd中输入

    node -v

    git --version

    hexo -v

    可以分别查看node.js, Git, hexo的版本，说明安装成功

  4. 注册Github账号并建立仓库

    登录后在主页点击新建仓库。仓库名称的格式必须为“用户名.github.io”。注意格式为“用户名.github.io”，否则无法显示博客界面。

  5. 设置ssh

    进入.ssh文件，输入邮箱。

    ssh-keygen -t rsa -C '邮箱'

    在C:\Users\lenovo.ssh目录下找到 id_rsa.pub 文件，使用记事本打开，复制生成的key。

    在Github主页的Settings中找到SSH and GPG keys，点击New SSH key。

    将刚才复制的key粘贴到这里。

    在本地验证成功。

    ssh -T git@github.com

    在本地绑定Github用户名和邮箱。

    git config --global user.name "用户名"

    git config --global user.email "邮箱"

    绑定成功后收到邮件。

  6. 上传博客

    使用记事本打开blog文件夹下的_config.yml文件，修改deploy配置，其中repository的内容可以在仓库的Code处查看。

    安装上传工具。

    cnpm install hexo-deployer-git

    新建一篇名为Hello的文章并重新生成一遍文件。

    hexo new "Hello"

    输入hexo s后，在浏览器输入localhost:4000，可以看到多了一篇名为Hello的文章。

    上传到github。

    hexo d

    上传成功后收到一封邮件。

    在Github仓库查看可以发现多了一些文件。

    此时在浏览器输入“用户名.github.io”即可访问博客。
